# ➕ Sumatoria de 100 Números: ¡Tu Primer Bucle `for`! 🚀

¿Listo para un nuevo desafío que te hará usar un poder fundamental en la programación: la repetición? En este ejercicio, vas a desarrollar un programa que calcule la suma total de cien números que el usuario ingrese. ¡Esta es la oportunidad perfecta para dominar la **estructura `for`**, una herramienta esencial para realizar tareas repetitivas de manera eficiente! 🧠💻

## ✨ ¿Qué debe hacer tu programa?

1.  **Solicitar 100 números:** El programa debe pedir al usuario que ingrese, uno por uno, un total de 100 números.
2.  **Acumular la suma:** A medida que el usuario ingresa cada número, el programa debe ir sumando estos valores en una variable acumuladora.
    * *Recuerda:* Deberás inicializar una variable en cero antes de comenzar la suma.
3.  **Mostrar el resultado:** Una vez que se hayan ingresado los 100 números, el programa debe mostrar la sumatoria total de todos ellos.

> 💡 Tip: La clave de este ejercicio es utilizar un **bucle `for`** que se repita exactamente 100 veces. Dentro de cada repetición del bucle, le pedirás al usuario que ingrese un número y lo sumarás a tu acumulador.

## 🧠 ¿Qué se practica en este ejercicio?

Este ejercicio es excelente para reforzar y aplicar varios conceptos clave:

-   ✅ **Estructuras de control repetitivas:** El uso fundamental del **bucle `for`** para ejecutar un bloque de código un número determinado de veces.
-   ✅ **Contadores e Iteradores:** Comprender cómo el `for` utiliza un contador para controlar las repeticiones.
-   ✅ **Variables acumuladoras:** Cómo declarar e ir actualizando una variable para guardar el resultado de una suma progresiva.
-   ✅ **Entrada de datos repetida:** Practicar la lectura de datos del usuario múltiples veces dentro de un bucle.
-   ✅ **Resolución de problemas con repetición:** Aplicar un patrón algorítmico común para procesar colecciones de datos.

## ✅ Conclusión

Este desafío es tu puerta de entrada al mundo de los bucles, una de las estructuras más poderosas y utilizadas en programación. ¡Dominar el bucle `for` te permitirá automatizar tareas y procesar grandes volúmenes de datos con facilidad, convirtiéndote en un programador más eficiente! 📈✨
